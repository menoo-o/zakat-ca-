# Zakat & FX Dashboard

A modern Islamic-fintech web application that combines a **Zakat Calculator** with **live Forex, Gold, and Silver rates** in a clean and minimal dark-themed interface.

Designed with a focus on simplicity, financial clarity, and premium user experience.

---

## ✨ Features

### 🧮 Zakat Calculator
- Calculate Zakat based on total assets and wealth
- Gold & Silver Nisab support
- Multiple currency support
- Clean and guided calculation flow
- Minimal and distraction-free UI

### 💱 FX & Metals Rates
- Live Forex exchange rates
- Gold (XAU) and Silver (XAG) tracking
- Currency conversion support
- Real-time market-focused dashboard
- Premium dark terminal-inspired design
